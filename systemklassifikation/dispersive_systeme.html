Dispersive Systeme

Verschiedene Frequenzen werden unterschiedlich verzögert

Anleitung

Ein Impuls enthält mehrere Frequenzen. In dispersiven Systemen

reisen hohe Frequenzen langsamer als niedrige - der Impuls zerfließt

τ(ω) ≠ const → Laufzeit τ hängt von Frequenz ω ab
Was ist Dispersion?
Ein dispersives System verzögert verschiedene Frequenzkomponenten unterschiedlich stark. Dies führt zur Verzerrung von Impulsen - das Signal zerfließt in der Zeit.

Beispiele: Glasfaserkabel, Atmosphäre, Wasserwellen, Prismen (Licht)
Höhere Werte = stärkere Trennung

Kernaussagen

  • Dispersive Systeme haben eine frequenzabhängige Laufzeit τ(ω)
  • Verschiedene Frequenzen werden unterschiedlich stark verzögert
  • Ein scharfer Impuls wird dadurch in der Zeit verschmiert
  • Problem bei Datenübertragung: Impulse überlappen → Intersymbol-Interferenz
  • In nicht-dispersiven Systemen: τ(ω) = const → keine Verzerrung
