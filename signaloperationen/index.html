Signaloperationen

Verarbeitung und Transformation von Signalen

Wichtige Konzepte

  • Faltung: y(t) = x(t) * h(t) = ∫ x(τ)·h(t-τ) dτ - Systemantwort auf beliebige Eingänge
  • Impulsantwort h(t): Vollständige Charakterisierung linearer zeitinvarianter Systeme
  • Faltungssatz: x(t) * h(t) ↔ X(ω)·H(ω) - Faltung im Zeitbereich = Multiplikation im Frequenzbereich
  • Kommutativität: x(t) * h(t) = h(t) * x(t) - Reihenfolge spielt keine Rolle
  • Assoziativität: (x * h₁) * h₂ = x * (h₁ * h₂) - Kettenschaltung von Systemen
  • Distributivität: x * (h₁ + h₂) = x * h₁ + x * h₂ - Parallelschaltung von Systemen
  • Neutrales Element: x(t) * δ(t) = x(t) - Dirac-Impuls ist neutral
Signaloperationen beschreiben, wie Signale mathematisch verarbeitet und transformiert werden. Die Faltung ist eine der wichtigsten Operationen in der Signalverarbeitung und beschreibt die Systemantwort auf beliebige Eingangssignale.
