Rechteckfunktion

rect(t/T) - Rectangular Pulse Function

Beschreibung

Die Rechteckfunktion ist ein fundamentaler Puls in der Signalverarbeitung.

Sie wird oft für Fensterung und Modulation verwendet.

rect(t/T) = { 1 für |t| < T/2, 0 für |t| ≥ T/2 }

Eigenschaften der Rechteckfunktion

  • Symmetrie: Gerade Funktion (rect(-t) = rect(t))
  • Normierung: Fläche = T (bei Amplitude 1)
  • Fourier-Transformation: F{rect(t/T)} = T·sinc(ωT/2π)
  • Beziehung zu Sprungfunktion: rect(t) = u(t+1/2) - u(t-1/2)
  • Anwendung: Fensterung, Zeitbegrenzung, Abtastung
