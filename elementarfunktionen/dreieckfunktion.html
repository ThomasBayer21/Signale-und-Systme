Dreieckfunktion

tri(t/T) - Triangular Function

Beschreibung

Die Dreieckfunktion ist eine stetige Elementarfunktion mit linearen Flanken.

Sie wird häufig in der Interpolation und als Fensterfunktion verwendet.

tri(t/T) = { (1 - |t|/T) für |t| < T, 0 für |t| ≥ T }

Eigenschaften der Dreieckfunktion

  • Stetigkeit: Stetig, aber nicht differenzierbar an den Knickpunkten
  • Symmetrie: Gerade Funktion (tri(-t) = tri(t))
  • Normierung: Fläche = T (bei Amplitude 1)
  • Fourier-Transformation: F{tri(t/T)} = T·sinc²(ωT/2π)
  • Beziehung: tri(t) = rect(t) * rect(t) (Faltung zweier Rechtecke)
  • Anwendung: Interpolation, Fensterung, lineare FM-Modulation
