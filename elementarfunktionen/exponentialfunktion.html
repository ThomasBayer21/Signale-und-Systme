Exponentialfunktion

e^(at)·u(t) - Einseitige Exponentialfunktion

Beschreibung

Die kausale Exponentialfunktion e^(at)·u(t) ist fundamental für die Beschreibung von Systemen erster Ordnung.

Für a < 0 beschreibt sie ein abklingendes Signal, für a > 0 ein ansteigendes Signal.

x(t) = A·e^(at)·u(t) mit a ∈ ℝ

Eigenschaften der Exponentialfunktion

  • Abklingen (a < 0): Signal klingt exponentiell ab, Zeitkonstante τ = -1/a
  • Ansteigen (a > 0): Signal wächst exponentiell an (instabil)
  • Ableitung: d/dt[e^(at)] = a·e^(at) (selbstähnlich)
  • Fourier-Transformation: F{e^(at)·u(t)} = 1/(jω - a) für a < 0
  • Laplace-Transformation: L{e^(at)·u(t)} = 1/(s - a)
  • Zeitkonstante: Bei t = τ = 1/|a| ist Signal auf 1/e ≈ 36.8% abgefallen
  • Anwendung: RC-Glied, Kondensatorentladung, Relaxationsprozesse, Systemantworten
