Sprungfunktion

Heaviside-Funktion u(t) oder σ(t)

Beschreibung

Die Sprungfunktion (Unit Step Function) ist eine fundamentale Elementarfunktion in der Signalverarbeitung.

Sie beschreibt einen idealen, instantanen Übergang von 0 auf 1.

u(t - t₀) = { 0 für t < t₀, 1 für t ≥ t₀ }

Eigenschaften der Sprungfunktion

  • Diskontinuität: Sprung bei t = t₀
  • Ableitung: u'(t) = δ(t) (Dirac-Impuls)
  • Integral: ∫u(t)dt = t·u(t) (Rampenfunktion)
  • Anwendung: Modellierung von Schaltvorgängen und Einschaltvorgängen
  • Fourier-Transformation: U(ω) = πδ(ω) + 1/(jω)
