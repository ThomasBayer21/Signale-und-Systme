Logarithmus-Eigenschaften

Warum Multiplikation zu Addition wird und andere wichtige Eigenschaften

Der Logarithmus komprimiert und vereinfacht

Der Logarithmus macht große Wertebereiche handhabbar und verwandelt komplizierte Rechenoperationen in einfachere

Grundlage der dB-Skala: dB = 10 · log₁₀(Verhältnis) oder dB = 20 · log₁₀(Verhältnis)

1. Produkt → Summe

log(a · b) = log(a) + log(b)
Beispiel: log(100) = log(10 · 10) = log(10) + log(10) = 1 + 1 = 2
In dB: 10× · 10× = 100× entspricht 20dB + 20dB = 40dB

2. Quotient → Differenz

log(a / b) = log(a) - log(b)
Beispiel: log(10) = log(100/10) = log(100) - log(10) = 2 - 1 = 1
In dB: 100×/10× = 10× entspricht 40dB - 20dB = 20dB

3. Potenz → Multiplikation

log(an) = n · log(a)
Beispiel: log(10³) = 3 · log(10) = 3 · 1 = 3
In dB: (10×)³ = 1000× entspricht 3 · 20dB = 60dB

4. Wurzel → Division

log(ⁿ√a) = log(a) / n
Beispiel: log(√100) = log(100) / 2 = 2 / 2 = 1
In dB: √(100×) = 10× entspricht 40dB / 2 = 20dB

5. Basis-Wechsel

loga(x) = logb(x) / logb(a)
Beispiel: log₂(8) = log₁₀(8) / log₁₀(2) ≈ 0.903 / 0.301 = 3
Umrechnung zwischen verschiedenen Logarithmus-Basen

6. Reziprok (Kehrwert)

log(1/a) = -log(a)
Beispiel: log(1/10) = -log(10) = -1
In dB: 1/10× = 0.1× entspricht -20dB

Warum der Logarithmus in der Signalverarbeitung?

  • Große Wertebereiche: Von 0.001 bis 1000000 werden zu -60dB bis +120dB
  • Menschliche Wahrnehmung: Unser Gehör empfindet logarithmisch (doppelte Lautstärke ≈ +10dB)
  • Kettenschaltungen: Verstärkungen multiplizieren sich → in dB einfach addieren
  • Frequenzgänge: Bode-Diagramme zeigen Verstärkung in dB über logarithmischer Frequenzachse
  • Genauigkeit: Relative Fehler werden zu absoluten Fehlern (±3dB statt ×1.4 oder ×0.7)
